Security

  • Keys are stored in chrome.storage.local — on this device only, never synced to the cloud.
  • Keys are only read inside the extension's background process and injected directly into API request headers.
  • Web pages and content scripts never have access to your keys.
  • Use API keys with spending limits set at the provider dashboard for additional protection.
